Html e CSS Box-model

Formatação de Box

  • height = Altura
  • width = Largura
  • border = Borda linha em volta do conteudo
  • padding = Acochoamento/preenchimento espaço interno
  • margin = Espaço externo
  • outline = contorno/traçado

Tipos de Caixa

  • box-leve = linha nova largura total da tela
  • inline-level= puxa pro lado
